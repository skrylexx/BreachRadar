# BreachRadar Backend

Ce dossier contient le backend FastAPI et le moteur BreachRadar (orchestrateur de scans, intégrations OSINT, génération de rapports).

Ce paquet est construit via `pyproject.toml` avec `hatchling` et est principalement destiné à être installé et exécuté dans le conteneur Docker.

Pour plus de détails sur l’architecture globale et la WebUI, voir le `README.md` à la racine du dépôt.
